1. 브랜딩

CEN AI의 브랜드 아이덴티티를 정의하는 핵심 요소들입니다.

1.1 로고

로고 사용 규칙

  • 최소 크기: 높이 64px (좌측 사이드바), 32px (일반 사용)
  • 클리어 스페이스: 로고 주변 최소 16px 여백 확보
  • 라이트 모드: 컬러 로고 사용 (logo-light)
  • 다크 모드: 화이트 로고 사용 (logo-dark)
  • 배경: 충분한 대비를 가진 배경 위에만 배치

권장사항

  • 충분한 여백 확보
  • 원본 비율 유지
  • 테마에 맞는 버전 사용

금지사항

  • 로고 변형 및 왜곡
  • 로고 색상 변경
  • 낮은 대비 배경 사용

1.4 톤앤매너

  • 전문적이지만 친근한 - 기술적이면서도 접근하기 쉬운 톤
  • 명확하고 간결한 - 복잡한 개념을 쉽게 설명
  • 혁신적이고 미래지향적 - AI 기술의 가능성 강조

작문 가이드

  • 능동태 사용 권장
  • 긍정적인 언어 사용
  • 전문 용어는 필요시에만, 설명 추가
  • 사용자 중심 언어 (당신, 여러분)

2. 레이아웃

일관된 레이아웃과 간격 체계를 통해 조화로운 UI를 구성합니다.

2.1 그리드 시스템

그리드 규칙

  • 최대 컨테이너 너비: 1400px (데스크톱)
  • 사이드바 너비: 280px (확장), 72px (축소)
  • 컬럼 갭: 24px ~ 32px
  • 그리드 타입: CSS Grid 사용 권장

2.2 간격 체계

4px 기반의 일관된 간격 체계를 사용합니다.

Spacing Scale
xs: 4px
sm: 8px
md: 16px
lg: 24px
xl: 32px
2xl: 48px
/* CSS Variables */ --spacing-xs: 4px; --spacing-sm: 8px; --spacing-md: 16px; --spacing-lg: 24px; --spacing-xl: 32px; --spacing-2xl: 48px;

2.3 브레이크포인트

디바이스 브레이크포인트 컨테이너 너비
모바일 (Small) < 640px 100%
태블릿 (Medium) 641px ~ 1024px 100%
데스크톱 (Large) 1025px ~ 1440px 1200px
대형 데스크톱 (XL) > 1440px 1400px

2.4 반응형 규칙

모바일 우선 (Mobile First)

모바일 레이아웃을 기본으로 하고, 더 큰 화면을 위한 스타일을 추가합니다.

/* 모바일 기본 */ .container { padding: 16px; } /* 태블릿 이상 */ @media (min-width: 768px) { .container { padding: 32px; } }

적응형 레이아웃

  • 사이드바: 데스크톱에서만 표시
  • 그리드: 자동으로 컬럼 조정
  • 폰트 크기: 작은 화면에서 축소
  • 터치 타겟: 최소 44x44px

4. 인터랙션

자연스러운 사용자 경험을 위한 애니메이션과 인터랙션 규칙입니다.

4.1 애니메이션 속도

속도 지속 시간 사용 케이스
Fast 0.15s 작은 UI 변화, 툴팁 표시
Base 0.2s 버튼 호버, 색상 변경, 토스트 표시
Slow 0.3s 드롭다운 열기, 모달 표시, 사이드바 확장
Slower 0.5s ~ 0.8s 페이지 전환, 복잡한 애니메이션
/* Transition Variables */ --transition-fast: 0.15s ease; --transition-base: 0.2s ease; --transition-slow: 0.3s ease; /* Usage */ .button { transition: all var(--transition-base); }

4.2 이징 (Easing)

기본 이징

  • ease - 일반적인 전환
  • ease-in - 시작 시 천천히
  • ease-out - 끝날 때 천천히 (권장)
  • ease-in-out - 시작과 끝 천천히

커스텀 이징

/* 부드러운 등장 */ cubic-bezier(0.16, 1, 0.3, 1) /* 바운스 효과 */ cubic-bezier(0.68, -0.55, 0.265, 1.55)

4.3 로딩 상태

Loading Spinner

로딩 인디케이터 사용 시기:

  • 1초 이상 걸리는 작업
  • 서버 요청 대기 중
  • 파일 업로드/다운로드

/* Loading Spinner */ .spinner { width: 40px; height: 40px; border: 4px solid var(--border-color); border-top-color: var(--accent-blue); border-radius: 50%; animation: spin 0.8s linear infinite; } @keyframes spin { to { transform: rotate(360deg); } }

5. 접근성

모든 사용자가 제품을 이용할 수 있도록 접근성 표준을 준수합니다.

5.1 컬러 대비율 (WCAG 2.1 기준)

레벨 일반 텍스트 큰 텍스트 설명
AA (최소) 4.5:1 3:1 법적 요구사항 충족
AAA (향상) 7:1 4.5:1 높은 접근성 보장

프로젝트 컬러 대비 검증

  • Primary Text (#1E1E1E) on White: 16.4:1 (AAA) ✓
  • Secondary Text (#6B7280) on White: 5.9:1 (AA) ✓
  • White on Blue (#0055E9): 5.8:1 (AA) ✓
  • Blue (#0055E9) on White: 5.8:1 (AA) ✓

5.2 키보드 네비게이션

필수 키보드 지원

  • Tab - 다음 요소로 이동
  • Shift+Tab - 이전 요소로 이동
  • Enter/Space - 버튼 활성화
  • Esc - 모달/드롭다운 닫기
  • Arrow keys - 목록/메뉴 탐색

포커스 관리

  • 모든 인터랙티브 요소 포커스 가능
  • 명확한 포커스 인디케이터
  • 논리적인 Tab 순서
  • 모달 열릴 때 포커스 트랩

6. 메시지 & 알림

사용자와의 효과적인 커뮤니케이션을 위한 다양한 메시지 타입과 사용 케이스입니다.

6.1 토스트 메시지 (Toast)

일시적인 피드백을 제공하는 경량 알림입니다. 3-5초 후 자동으로 사라집니다.

Toast Examples
자동 저장되었습니다
연결이 불안정합니다
파일 업로드 실패
자동 저장중...

토스트 사용 케이스

  • Success: 자동 저장 완료, 스토리보드 생성 완료, 프로젝트 저장 성공
  • Warning: 네트워크 불안정, 세션 만료 임박, 한도 초과 경고
  • Error: 이미지 업로드 실패, 서버 연결 실패, 권한 없음
  • Info: AI 생성 진행중, 자동 저장중, 업데이트 안내

6.2 알림 메시지 (Alert)

페이지 내에서 중요한 정보를 강조하는 박스형 메시지입니다.

Alert Examples
스토리보드가 성공적으로 생성되었습니다
8개의 씬이 생성되었습니다. 각 씬을 확인하고 수정할 수 있습니다.
무료 플랜 한도 임박
이번 달 생성 가능한 프로젝트가 2개 남았습니다. 플랜을 업그레이드하시겠습니까?
필수 정보를 입력해주세요
프로젝트 이름과 타겟 고객은 필수 입력 항목입니다.
AI 생성 시간 안내
스토리보드 생성에는 약 2-3분이 소요됩니다. 잠시만 기다려 주세요.

알림 사용 케이스

  • Success: 작업 완료 확인, 스토리보드 생성 완료
  • Warning: 플랜 한도 임박, 저장하지 않은 변경사항
  • Error: 폼 검증 실패, 필수 정보 누락, 파일 형식 오류
  • Info: AI 생성 진행 안내, 기능 사용 방법

7.3 이미지 비율 및 처리

  • 프로필 이미지: 1:1 (정사각형)
  • 배너 이미지: 21:9 (ultra wide)

이미지 최적화

  • Lazy loading 적용
  • 품질: 80-90%
  • 대체 텍스트(alt) 필수
